  • Basic HTML (classrooms) - Monday am
    HTML
    Allows coders to structure graphical and text content in a web page.
    In this lesson

    You will learn how to code a web page from scratch.

    1. Teacher and students pre-read lesson instructions together
    2. Teacher shows tutorial video to students
    3. Students complete basic lesson (Thor or Captain Marvel) with their partner
    4. Students adapt lesson code to make it cool (i.e., more images, text etc.)!
    5. Student groups show pages to peers and teachers for feedback and points
